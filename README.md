# 🕷️ Homem-Aranha | Landing Page

Projeto de página web desenvolvido com **HTML + CSS**, com foco em **layout moderno, organização e boas práticas de front-end**.

---

## 📌 Sobre o Projeto

Esta página apresenta o universo do **Homem-Aranha**, abordando:

* Origem de Peter Parker
* Habilidades do herói
* Dualidade entre vida pessoal e heroica
* Desafios e perdas
* Legado e mensagem final

O objetivo foi criar uma **landing page visualmente profissional**, com identidade inspirada em **quadrinhos (HQ)**.

---

## 🎨 Design e Estilo

* 🎯 Tema baseado em **vermelho, azul e preto**
* 🧱 Bordas grossas (estilo quadrinhos)
* 📦 Sombras duras (efeito HQ)
* 🎭 Layout moderno com seções bem definidas
* 🖼️ Imagens com alto contraste e saturação
* ✨ Efeitos de hover no menu
* 📜 Scroll suave entre seções

---

## 🧩 Estrutura da Página

* **Hero Section** (cabeçalho com destaque)
* **Origem**
* **Habilidades (cards)**
* **Dualidade (duas colunas)**
* **Galeria de desafios**
* **Citação em destaque**
* **Conclusão / Legado**
* **Rodapé**

---

## ⚙️ Tecnologias Utilizadas

* HTML5
* CSS3
* Flexbox
* Grid Layout

---

## 🚀 Funcionalidades

* Navegação por âncoras
* Scroll suave (`scroll-behavior`)
* Efeitos de hover com animação
* Layout responsivo básico
* Organização em containers

---

## 🧠 Aprendizados

Durante o desenvolvimento foram aplicados conceitos como:

* Estruturação semântica com HTML
* Organização de CSS (clean code)
* Uso de classes e seletores
* Flexbox para alinhamento
* Grid para galerias
* Estilização com identidade visual consistente
